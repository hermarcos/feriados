# Feriados

> **Note:** This project is part of the *Introducción a TDD* course by 10Pines Academy.  
> The code follows the exercises from the course and is intended for learning purposes.

**Feriados** is a Java project to learn **Test-Driven Development (TDD)** through a practical example: a **calendar of holidays**. The exercises follow the course step by step, providing hands-on experience with TDD principles.
